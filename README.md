# Java JSP Mysql Ticket Sales
Bu proje Java, JSP yazılımlarını ve Mysql database'ni kullanarak bir bilet firmasının bütün ihtiyaçlarını karşılayan bir web uygulaması sunar.
                    
                    
IDE and Tools | |
------------- | -------------
Intellij Ultimate | Java 8
Mysql | JSP
CSS3 | Jquery
